TodoApp(WebApp)

I built a full-stack Todo app using Python, FastAPI, PostgreSQL, and Jinja Templating. The app features JWT-based authentication for secure user access and supports complete CRUD functionality for task management. I utilized Jinja to create dynamic and user-friendly interfaces, while styling the front-end with HTML and CSS for a visually appealing design. Additionally, I extensively tested and validated the APIs using Postman to ensure everything works seamlessly.
